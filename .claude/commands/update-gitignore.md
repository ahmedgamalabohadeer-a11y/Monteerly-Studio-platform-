# /update-gitignore

Use this workflow when working on **update-gitignore** in `Monteerly-Studio-platform-`.

## Goal

Adds or updates the .gitignore file to control which files and directories are tracked by git.

## Common Files

- `.gitignore`

## Suggested Sequence

1. Understand the current state and failure mode before editing.
2. Make the smallest coherent change that satisfies the workflow goal.
3. Run the most relevant verification for touched files.
4. Summarize what changed and what still needs review.

## Typical Commit Signals

- Edit or create .gitignore file
- Commit .gitignore changes
